# WPF - User Control, buttons and ClickButton.

Example of how to change between diferent user controls with buttons help.

## Getting Started

You need Visual Studio 2019 to run this project. 

### Prerequisites

.NET Framework 4.7.2

##Built With

Build and run with Visual Studio 2019. 

- Select project "WPFUserControlsClickButton" and click StartUP Project.
- Select debug´s target and press Start.
